Molecular Flipbook

We are all visual learners. Seeing something in motion is far more memorable than reading a description of it. Molecular Flipbook is an animation software designed for molecular biologists to animate protein interaction using an intuitive interface.

This project is made possible by a grant from the NSF. The goal of the software

Because the software is built on top of Blender, it was pretty easy to make it cross-platform. However, many of the scientific packages still have to be compiled for each platform for optimal performance, so Linux support is still work-in-progress.
